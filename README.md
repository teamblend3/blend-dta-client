## 시작가이드

### Requirements

- Node.js 18.16.0
- Npm 9.5.1

### Installation

#### Backend

```bash
$ cd blend-dta-server
$ nvm use v.18.16.0
$ npm install
$ npm run dev
```

#### Frontend

```bash
$ cd blend-dta-client
$ nvm use v.18.16.0
$ npm install
$ npm run dev
```
